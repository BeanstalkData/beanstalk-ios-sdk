Tracking
  • getTransactions(contactId:startDate:endDate:handler:)

    Retrieve transaction events.

    • Dates are sent in the format YYYY-MM-DD.
    • If no dates are passed, all events for the contactId will be returned.
    • To retrieve events for a specific date, set startDate and endDate to the same date.
    • If startDate and endDate are both set, all events in that range will be returned.
    • If only startDate is set, all events from that day until now will be returned.

    • If only endDate is set, all events until that day will be returned.
